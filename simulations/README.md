# Simulations

This folder contains the computational prototypes used to generate the Coherence Under Constraint (CUC) figure set.

## Files

- `cuc_simulation_v1_coherence_emergence.py`  
  Generates time-domain coherence emergence across multiple CUC regimes.

- `cuc_simulation_v2_phase_diagram.py`  
  Generates the coupling–constraint phase diagram.

- `cuc_simulation_v3_survival_maps.py`  
  Generates the throughput–dissipation survival map and survival boundary.

- `cuc_simulation_v4_unified_figure.py`  
  Combines all prior simulation outputs into a unified computational figure plate.

## Purpose

These scripts provide a computational demonstration of the CUC framework by modeling coherence emergence, stability regions, energy constraints, and structural persistence.
